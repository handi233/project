#Sistem pendaftaran Pasien berbasis web

Aplikasi web untuk memudahkan pendaftaran pasien baru dan lama
secara online di klinik. Sistem ini membantu pasien mendaftar tanpa harus langsung ke klinik, sekaligus memudahkan staf klinik dalam mengelola data pendaftaran.

##Fitur Utama
-Pendaftaran pasien baru dan pasien lama
-Formulir input data pasien
-Notifikasi pendaftaran berhasil
-Dasbor informasi company profil klinik

#Teknologi yang digunakan
-Bahasa pemograman : PHP
-Framwork :PHP Native
-Database: My sql
-Frontend:HTML, CSS, JavaScript

##Cara Menjalankan 
1.Clone repository ini git clone https://github.com/handi233/p


2. Setup database MySQL dan import file database 

3.Konfigurasi file koneksi database sesuai config local

4. Jalankan aplikkasii dengan server local seperti XAMPP atau LAMP
